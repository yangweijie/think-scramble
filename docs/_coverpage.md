# ThinkScramble

> 🚀 ThinkPHP OpenAPI 文档生成器

- 🎯 **零配置启动** - 安装即用，无需复杂配置
- 📝 **自动文档生成** - 无需手动编写 PHPDoc 注解
- ⚡ **高性能缓存** - 智能缓存机制，支持增量解析
- 🎨 **现代化 UI** - 支持 Swagger UI 和 Stoplight Elements
- 🛡️ **安全分析** - 自动识别中间件和安全方案
- 📊 **OpenAPI 3.0** - 完全符合 OpenAPI 3.0 规范
- 🔌 **插件系统** - 可扩展的插件架构
- 💻 **CLI 工具** - 强大的命令行工具
- 🥧 **PIE 支持** - 现代化包管理体验
